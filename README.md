# android-slide-to-unlock
imitate ios, slide to unlock text color gradient animation

这是一个模仿苹果"slide to unlock"动画的demo应用。<br>

代码比较简单，就是用android自带的着色器shader去drawText，根据不同的参数，设置着色器不同的着色位置。<br>
